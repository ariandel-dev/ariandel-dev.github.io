크롬에서 잘 작동하던 모달이 사파리에서 안보이길래 확인해보니 높이가 0이었다.

인터넷을 다뒤져봐도 해당이슈에대한 언급이 없어서 새창을 열어보니 제대로 작동했다..

---

모달 랜더링을 디버깅하다가 nextjs-portal이라는 태그의 엘리먼트가 생성되어 있는것을 발견했다.

Next.js에서 다이얼로그, 모달 같은 UI 요소는 일반적인 div 안에 렌더링하면 레이아웃이 깨질 수 있어서
포털 기능을 사용하여, 모달 창을 DOM 트리의 최상위(root) 레벨에 렌더링한다고 한다.
