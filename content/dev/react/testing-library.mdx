nuqs패키지 기여 가이드 [문서](https://testing-library.com/docs/)를 확인하다가 링크를 타서 들어갔는데 좋은 [포스트](https://kentcdodds.com/blog/testing-implementation-details)를 발견해서 메모해둔다.

리액트 ui테스트용 패키지는 저거 쓰면될것같다. 제공하는 api들만 잘 사용하면 구현 세부사항에 대한 테스트를 자연스럽게 지양할 수 있을 것 같다.
