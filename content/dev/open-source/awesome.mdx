cursor를 쓰면서 rule이라는 기능을 찾았는데 프롬프트에 컨텍스트를 추가할때 사용하는 기능같다.
이런건 분명 깃허브에 좋은게 있을것 같아서 찾아보았더니 [있었다](https://github.com/PatrickJS/awesome-cursorrules).

태그에 awesome이 붙어있길래 링크를 타고가니까 스타수 30만짜리 [레포](https://github.com/sindresorhus/awesome)를 찾았다.

분야별로 트랜디한 내용을 정리해놓은 것 같은데 자주 둘러볼 것 같아서 메모로 남긴다.
