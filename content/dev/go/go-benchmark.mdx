### benchmark function

go에서는 벤치마크용 함수도 작성가능하다.

`go test -bench=.` 명령어로 벤치마크 테스트를 할 수 있다.

주의점으로는 벤치마크는 보통 실행시간이 안정될때까지 반복해서 실행되는데 다음과 같은 경우 
`n`이 증가할 수록 실행시간이 길어지기 때문에 벤치마킹이 안끝난다..

```go
func BenchmarkFibWrong(b *testing.B) {
    for n := 0; n < b.N; n++ {
        Fib(n)
    }
}
```
